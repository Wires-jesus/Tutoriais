# Mapa de Fluxo de Merge - ADMDB PL/SQL

## Introdução

Este documento mapeia o fluxo de carga baseado em **MERGE** no repositório ADMDB-plsql-MAS.
Para cada tabela que recebe dados, este mapa identifica qual(is) objeto(s) PL/SQL (Procedure, Package, Function ou Trigger) realiza(m) a operação de merge/insert.

**Propósito:** Entender o fluxo de alimentação de dados nas tabelas e identificar rapidamente qual objeto é responsável por manter cada tabela atualizada.

## Estatísticas

- **Total de Tabelas:** 528
- **Total de Mapeamentos:** 849
- **Objetos envolvidos:** Procedures, Packages, Functions e Triggers

## Como Usar Este Mapa

1. **Localize a tabela** que você precisa entender na lista abaixo (em ordem alfabética)
2. **Identifique o objeto** (Procedure, Package, etc.) responsável por alimentá-la
3. **Navegue até o arquivo** do objeto no repositório para entender a lógica completa
4. **Coluna Tipo** mostra se é MERGE ou INSERT simples
5. **Coluna Contagem** indica quantas vezes esse objeto interage com a tabela
